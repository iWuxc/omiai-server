# 微信数据导入指南

## 1. 概述
本系统支持通过粘贴文本的方式，批量导入微信中的客户资料。您无需手动逐条录入，只需将聊天记录导出或复制出来，系统即可自动识别关键信息。

## 2. 如何获取数据

### 方法 A: 电脑端微信直接复制 (推荐)
1. 登录电脑端微信 (Windows/Mac)。
2. 打开与客户的聊天窗口，或者打开您汇总信息的“文件传输助手”。
3. 选中包含客户信息的文本段落。
   * 建议格式：
     ```
     姓名：张三
     电话：13800138000
     身高：175
     ...
     ```
4. 右键点击“复制”或按 `Ctrl+C` (Mac: `Cmd+C`)。
5. 在系统的“批量数据导入”页面，粘贴到输入框中。

### 方法 B: 聊天记录迁移
如果您有大量分散的聊天记录，建议先在微信中搜索关键词（如“姓名”），然后将搜索结果中的有效信息逐条转发给“文件传输助手”，最后一次性复制文件传输助手中的内容。

## 3. 支持的字段格式
系统采用智能模糊匹配，支持以下常见格式：

* **姓名**: `姓名: xxx`, `名字：xxx`
* **电话**: `电话: 138xxxx`, `手机：139xxxx` (自动提取11位数字)
* **性别**: `性别: 男/女`
* **出生年月**: `生日: 1990-01`, `年份：95年` (自动转为 YYYY-MM)
* **身高/体重**: `身高: 175cm`, `体重: 60kg` (自动提取数字)
* **学历**: `学历: 本科`, `博士`
* **收入**: `收入: 10000`, `月薪：2万` (尝试提取数字)

## 4. 注意事项
* **去重规则**: 系统会根据“手机号”自动去重。如果手机号已存在，该条记录将跳过导入。
* **分批建议**: 为保证浏览器性能，建议每次粘贴 50-100 条记录（约 5000-10000 字）进行解析。
