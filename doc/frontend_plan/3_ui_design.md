# UI 设计规范与原型描述

## 1. 设计风格定义 (Visual Identity)
- **关键词**：专业、温暖、高效、信任。
- **主色调 (Brand Color)**：
    - **浪漫红 (#FF4D6A)**：用于主按钮、关键图标、选中状态（象征姻缘）。
    - **辅助金 (#FFB800)**：用于VIP标识、高亮标签。
- **背景色**：
    - 页面背景：#F5F6FA (浅灰，减少视觉疲劳)。
    - 卡片背景：#FFFFFF (纯白，突出内容)。
- **字体**：
    - 标题：#333333, 加粗, 16-18px。
    - 正文：#666666, 常规, 14px。
    - 辅助信息：#999999, 12px。

## 2. 核心页面原型描述

### 2.1 首页/工作台 (Home)
- **顶部**：
    - 问候语 + 今日日期。
    - 数据概览卡片（Grid布局）：`客户总数`、`今日新增`、`待匹配`。
- **中部**：
    - 快捷功能区（图标入口）：`录入档案`、`分享邀请`、`我的任务`。
- **底部**：
    - 待办事项列表（简略版）。

### 2.2 客户资源池 (Client List)
- **顶部搜索栏**：悬浮固定，支持姓名/手机号搜索。
- **筛选吸顶条**：`性别` | `年龄` | `学历` | `更多筛选+`。
- **列表区域**：
    - **卡片式布局**。
    - 左侧：圆形头像（若无照片显示默认性别图）。
    - 中间：姓名 + 性别图标 + 年龄。
    - 下方标签行：`本科` `175cm` `年入20w` `有房`。
    - 右侧：状态标签（待匹配/已匹配）。
    - *交互*：点击跳转详情，长按弹出快捷菜单（备注/删除）。

### 2.3 客户详情页 (Profile Detail)
- **头部**：大头像背景模糊 + 个人基本信息悬浮卡片。
- **Tab切换**：`基本资料` | `择偶标准` | `匹配推荐`。
- **内容区**：
    - 分组展示信息（基础、家庭、经济、独白）。
    - 照片墙：九宫格展示生活照，支持点击大图预览。
- **底部操作栏 (Fixed)**：
    - `编辑` | `生成海报` | `一键匹配` (高亮主按钮)。

### 2.4 信息采集页 (Form Collection - User Side)
- **设计原则**：降低填写心理负担。
- **形式**：分步向导 (Wizard)。
    - **Step 1 基础信息**：姓名、性别、生日（滚轮选择）、身高。
    - **Step 2 社会属性**：学历、工作、收入、房车（单选/多选按钮，减少打字）。
    - **Step 3 自我介绍**：家庭情况、独白（提供快捷标签点选）。
    - **Step 4 择偶要求**：范围选择器（如年龄范围 25-30）。
    - **Step 5 照片上传**：支持裁剪。
- **反馈**：提交成功后显示“红娘已收到，正在为您审核”。

## 3. 组件库选型
推荐使用 **uView UI 2.0**，其内置组件完全满足上述需求：
- `u-form`：强大的表单验证。
- `u-upload`：图片上传。
- `u-dropdown`：下拉筛选菜单。
- `u-tag`：多样的标签样式。
- `u-steps`：分步条。
